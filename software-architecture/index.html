Serverless, c'est quoi ?

  • On se focus uniquement sur la construction de notre application sans se soucier de l'infrastructure
  • Externalisation des services via des plateformes tiers
  • Au lieu d'exécuter notre code sur notre serveur, on l'exécute dans le cloud (FaaS)

Création d'un site pour le club studio

  • Seuls les étudiants de l'école peuvent se connecter au site (authentification)
  • On doit pouvoir liker, tagger des médias (database)
  • Les médias doivent être accessibles rapidement (storage)
  • Le tout serverless (site static + framework frontend)
